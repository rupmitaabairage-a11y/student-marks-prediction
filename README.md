# 🎓 Student Marks Prediction
This is a Machine Learning project where I have predicted the potential exam scores (Marks) of students based on their study hours.

---

### 📋 Project Features:
* **Algorithm:** I have used **Linear Regression** for this project.
* **Data:** A dataset was used showing how many marks are obtained based on study hours.
* **Goal:** When 'Study Hours' are provided as input, the model outputs the predicted marks.

### 🛠️ Technology and Libraries:
I used the following tools for this project:
* **Python**
* **Pandas** (For data analysis)
* **Scikit-learn** (For building the model)
* **Matplotlib** (For data visualization/graphs)

### 🚀 How to run:
1. Clone this repository.
2. Open the `students_mark_predictor.ipynb` file in Jupyter Notebook or Google Colab.
3. Run all the cells.
